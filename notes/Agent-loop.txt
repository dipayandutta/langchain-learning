KEY COMPONENTS OF THE LOOP
---------------------------

1. **QUERY**: User's input/question
   - Starting point of the loop
   - Contains the task to be solved

2. **THOUGHT**: Agent's reasoning step
   - Analyzes the query
   - Plans the approach
   - Decides if tools are needed

3. **ACTION**: Tool execution step
   - Calls appropriate tools
   - Performs the planned action
   - Only happens if tools are needed

4. **OBSERVATION**: Result from tool
   - Receives tool output
   - Processes the information
   - Feeds back to thinking

5. **DECISION POINTS**:
   - Tool needed? (YES/NO)
   - Task complete? (YES/NO)

6. **LOOP CONTROL**:
   - Continues until task is complete
   - Can make multiple tool calls
   - Adapts based on observations


LOOP TERMINATION CONDITIONS
---------------------------

The agent loop ends when:
- Task is fully completed
- Maximum iterations reached
- Error occurs that cannot be recovered
- User stops the process

This structured approach ensures agents can handle complex tasks by breaking them down into manageable steps.
