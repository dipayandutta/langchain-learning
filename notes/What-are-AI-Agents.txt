What is an AI Agent ?
=======================

An Agent is a software system that uses LLM's as a resoning engine
to decide what action to take next.

ReAct Agent Architecture
==========================

ReAct stands for **Re**asoning and **Act**ing - a framework that combines 
reasoning (thinking) and acting (doing) in a loop.

ReAct Pattern:
--------------
1. **Thought** - Reason about what to do next
2. **Action** - Choose and execute a tool/action  
3. **Observation** - Review the result of the action
4. **Repeat** - Continue until task is complete

Example ReAct Conversation:
---------------------------

User: "What's the current weather in New York and should I bring an umbrella?"

THOUGHT 1: "I need to find the current weather in New York City"
ACTION 1: "Search for current weather in New York"
OBSERVATION 1: "Temperature: 65°F, Rain: 80% chance, Wind: 15 mph"

THOUGHT 2: "Based on the weather data, there's an 80% chance of rain. I should recommend bringing an umbrella."
ACTION 2: "Provide recommendation based on weather data"
OBSERVATION 2: "Final answer prepared"

FINAL ANSWER: "It's 65°F in New York with an 80% chance of rain. Yes, you should definitely bring an umbrella!"
